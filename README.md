# [pickr.live](https://pickr.live)
This project allows you to randomly select an item from a list at a future date and time.

It's useful for scenarios like:
- Choosing a restaurant
- Selecting a movie to watch
- Giveaway / Raffle drawings

Pickr uses [drand](https://www.drand.love/) randomness beacons to ensure fair, consistent, and verifiable selections.

Pickr is not a mature project and is probably chock-full of bugs; feel free to squash any.
